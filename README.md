# K8sMCPs

This repository contains a collection of Kubernetes deployments of stand-alone Model Context Protocol (MCP) servers in Kubernetes.


## Available MCP Servers

- **[GitHub MCP Server](./github-mcp-server/)** - Official GitHub integration enabling AI tools to interact with repositories, issues, pull requests, workflows, and more

## Contributing

This repository focuses on deploying MCP servers in Kubernetes. Contributions are welcome for:

- New MCP server deployments
- Improved configurations
- Documentation enhancements
- Bug fixes
